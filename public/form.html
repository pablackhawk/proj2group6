UCI coding boot camp project #2

This project is incorporating the use of the Materialize front-end framework, along with JQuery and JavaScript for UX/UI. The site is served up with Node.js, Express and Sequalize for routing, requests, MVC and ORM functionality/methodologies. API's used are amazon, youtube and "". We are also using npm to require in various packages like request, method-override, body-parser and cheerio. We've also utilized web scraping for data extraction.
