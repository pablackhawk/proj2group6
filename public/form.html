UCI Coding Boot Camp Project #2

This project incorporates the use of the Materialize front-end framework, along with JQuery and JavaScript for UX/UI. The site is served up with Node.js, Express and Sequelize for routing, requests, MVC and ORM functionality/methodologies. APIs used are Amazon, YouTube and "". Node packages include request, method-override, body-parser and cheerio. Web Scraping is utilized as well for data extraction.
